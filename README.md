# ci-docker-stack
Docker Compose builds Jenkins , Sonar, portainer, mediawiki

Pre-Requisites
-------------
- Docker (for Mac OSX / Windows / Docker)
- Docker Compose
- Clone the repository

QuickStart
-----------
- To Start containers in the background

	`` 
	$ docker-compose up -d
	``
- Stoping all containers

	`` 
	$ docker-compose down
	``
